Keyboard Shortcuts & Gestures

Action Shortcut / Gesture
Pan Canvas Hold Space + Left-Click & Drag
Zoom In / Out Mouse Scroll Wheel (Zoom to mouse)
Select Tool Press V
Connect Tool Press C
Delete Box(es) Press Delete or Backspace
Multi-Select Hold Shift + Click boxes or Drag marquee select
Edit Box Label Double-click Box on canvas
Undo Action Press Ctrl + Z
Redo Action Press Ctrl + Y or Ctrl + Shift + Z
Spawn Child Box Press Tab (spawns connected box in next column)
Spawn Sibling Box Press Enter (spawns box in same column)
